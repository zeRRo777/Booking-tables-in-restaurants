AUTH:
    POST /api/auth/register
    Описание: Регистрация нового пользователя (Гостя). После успешной регистрации пользователь становится Клиентом.
    Доступ: Все (неаутентифицированные пользователи).
    Тело запроса: email, name, phone, password, password_confirmation

    POST /api/auth/login
    Описание: Аутентификация пользователя и получение токена доступа (JWT или Sanctum).
    Доступ: Все (неаутентифицированные пользователи).
    Тело запроса: email, password

    POST /api/auth/logout
    Описание: Выход пользователя из системы (аннулирование токена).
    Доступ: Все (аутентифицированные пользователи).

    POST /api/auth/password/change
    Описание: Инициация смены пароля.
    Доступ: Все (аутентифицированные пользователи).
    Тело запроса: curent_password, new_password.

    POST /api/auth/password/reset
    Описание: Запрос сброса пароля.
    Доступ: Все (аутентифицированные пользователи).
    Тело запроса: email.

    POST /api/auth/password/reset/confirm
    Описание: Подтверждение сброса пароля.
    Доступ: Все.
    Тело запроса: reset_token, new_password.

    POST /api/auth/email/change
    Описание: Инициация смены email.
    Доступ: Все (аутентифицированные пользователи).
    Тело запроса: new_email.

    POST /api/auth/email/change/confirm
    Описание: Потвержденеи смены нового email.
    Доступ: Все.
    Тело запроса: verification_code.

    POST /api/auth/phone/change
    Описание: Инициация смены phone.
    Доступ: Все (аутентифицированные пользователи).
    Тело запроса: new_phone.

    POST /api/auth/phone/change/confirm
    Описание: Потвержденеи смены phone.
    Доступ: Все.
    Тело запроса: verification_code,new_phone.

USERS:
    GET api/users
    Описание: все пользователи c сортировкой, фильтром и пагинацией.
    Доступ: Глобальный админ.
    Параметры запроса: email, name, phone, is_blocked, created_at_from, created_at_before

    GET api/user/{id}
    Описание: Информация об одном пользователи.
    Доступ: Глобальный админ.

    PATCH api/user/{id}
    Описание: Обновление данных пользователя.
    Доступ: Глобальный админ.
    Тело запроса: email, name, phone, is_blocked.

    DELETE api/user/{id}
    Описание: Удаление пользователя.
    Доступ: Глобальный админ.

    GET /api/users/{id}/reservations
    Описание: Просмотр бронирований конкретного пользователя
    Доступ: Глобальный админ, Админ ресторана (только для бронирований в его ресторане).

    GET api/me

    DELETE api/me
    Описание: Удаление данных профиля.
    Доступ: Аутентифицированный юзер.
    Тело запроса: name.

    GET api/me/reservations
    Описание: Получение списка своих бронирований.

    GET api/me/reviews
